# Pseudocódigo Cátedra 🎓

![Visual Studio Marketplace Version](https://img.shields.io/visual-studio-marketplace/v/tu-publisher.pseudocodigo-catedra?style=flat-square)
![License](https://img.shields.io/github/license/tu-usuario/tu-repo?style=flat-square)

Soporte de sintaxis, autocompletado y formateo básico para la variante de pseudocódigo utilizada en entornos académicos y cátedras de programación. Diseñado específicamente para resolver la falta de herramientas en la correcta visualización de refinamientos secuenciales y bloques estructurados en español.

## ✨ Características

*   **Resaltado de Sintaxis Académica:** Coloreado preciso para palabras clave (`si`, `mientras`, `funcion`), funciones nativas (`leer`, `escribir`) e identificadores de refinamiento (`A0`, `A1`, etc.).
*   **Snippets Inteligentes:** Autocompletado automático de estructuras de control con tabulación interna presionando solo `Tab`.
*   **Configuración del Editor:** Auto-cierre de bloques, paréntesis, llaves y strings.

## 🚀 Instalación

1. Abrí **Visual Studio Code**.
2. Presioná `Ctrl+Shift+X` para abrir el panel de Extensiones.
3. Buscá `Pseudocodigo Catedra`.
4. Hacé clic en **Install**.
